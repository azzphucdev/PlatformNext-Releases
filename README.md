## Overview

**Platform Next** (App Store: *Platform Next - Unlimited VPN*) is a polished, cross-platform proxy client that puts every major proxy engine and protocol behind a single, one-tap interface. Whether you want instant connectivity or full hand-crafted control, Platform Next adapts to you — with an adaptive glass UI, light and dark themes, and multi-language support.

> **This repository hosts RELEASES and the ISSUE TRACKER only.**
> It provides downloadable release binaries (APK, DMG, EXE, and more) and the public issue tracker for Platform Next. The application **source code is private / closed-source** and is **not** published here.

## Why Platform Next

- **One app, every engine.** Instead of juggling a separate client per protocol, Platform Next brings the leading proxy cores together behind a single, consistent interface.
- **Truly cross-platform.** The same app, the same experience, on Android, iOS, macOS, Windows, and Linux.
- **Simple by default, powerful when you need it.** One-tap connect for everyday use; a full Custom mode for hand-crafted and imported configurations.
- **Built for restricted markets.** Region-aware endpoints keep the app usable in tough network environments.
- **Privacy-first.** No user-activity logging and a zero data collection stance.

## Features

### One tap or full control
- **Home mode** — one-tap connect for instant, effortless connectivity.
- **Custom mode** — an advanced workspace for hand-crafted and imported configurations.
- **Invitation mode** and **V2Board subscription-account mode** — subscription management with auto-update.

### Import anything
- Import via **share-link**, **subscription URL**, **QR code**, or **raw config**.

### Powerful networking
- **Routing rules** / rule-based proxy for granular control.
- **Per-app proxy** (Android) and **custom DNS**.
- **Android root mode** with TUN + TPROXY transports.

### Built-in tools
- **Speed test**, **reachability / latency test**, **connection timer**, and live **traffic stats**.

### Made to feel great
- Adaptive **glass UI**, **light + dark** themes, and **multi-language** support (English, Vietnamese, and more).

## Cores & Protocols

Platform Next is a frontend for the industry's leading proxy engines — pick the right tool for any network.

### Cores / Engines

| Core / Engine | Role |
|---------------|------|
| **sing-box** | Default core |
| **Xray-core** | Proxy core |
| **v2ray-core** | Proxy core |
| **Mihomo** (a.k.a. Clash.Meta) | Proxy core |
| **OpenSSH / OSSH4** | Obfuscated SSH |
| **Psiphon** | Circumvention core |

### Supported Protocols

Across the six bundled cores (sing-box · Xray · v2ray-core · Mihomo · OSSH4 · Psiphon), Platform Next speaks **30+ protocols**:

| | | |
|---|---|---|
| VLESS (REALITY / XTLS-Vision) | VLESS-Encryption (ML-KEM) | VMess |
| Trojan | Shadowsocks | Shadowsocks-2022 |
| ShadowsocksR | Hysteria2 | Hysteria |
| TUIC | AnyTLS | ShadowTLS |
| NaiveProxy | Snell | Mieru |
| VLite | WireGuard | WARP |
| SSH | OSSH4 (Obfuscated SSH) | Psiphon |
| Tor | SOCKS4 / 4a / 5 | HTTP / HTTPS |

**Psiphon** and **OSSH4** each bundle multiple obfuscated tunnel protocols, chosen automatically or by hand:

| Core | Tunnel protocols |
| :-- | :-- |
| **Psiphon** | SSH · OSSH · Unfronted-Meek (OSSH / HTTPS / Session-Ticket) · Fronted-Meek (OSSH / HTTP) · QUIC-OSSH · Fronted-Meek-QUIC · Conjure-OSSH |
| **OSSH4** | Direct · HTTP-Proxy · SSL/TLS · WebSocket · DNSTT |

### Transports & Security

The bundled cores also cover the full modern transport, TLS and obfuscation stack:

| Category | Supported |
| :-- | :-- |
| **Transport networks** | TCP / Raw · WebSocket · gRPC · HTTP/2 · HTTPUpgrade · XHTTP (SplitHTTP) · mKCP · QUIC |
| **TLS & anti-censorship** | TLS · uTLS fingerprints · REALITY · XTLS-Vision · ShadowTLS |
| **Obfuscation & tuning** | Salamander obfs · multiplexing (mux / smux) · TCP Fast Open |

## Getting Started

### Android
1. Install from [Google Play](https://play.google.com/store/apps/details?id=dev.platformnext.vpn), or download the latest **APK** from the [Releases page](https://github.com/azzphucdev/PlatformNext-Releases/releases).
2. Open the app and grant the VPN permission when prompted.
3. Import a config via share-link, subscription URL, QR code, or raw config — or use Home mode for one-tap connect.

### iOS
1. Install from the [App Store](https://apps.apple.com/app/id6741474000).
2. Allow the VPN configuration when iOS prompts.
3. Add your config and tap connect.

### macOS
1. Download the **DMG** or **PKG** from the [Releases page](https://github.com/azzphucdev/PlatformNext-Releases/releases) — universal (Intel + Apple Silicon). On Apple Silicon Macs you can alternatively install the iOS app from the [App Store](https://apps.apple.com/app/id6741474000).
2. Open the app and approve the system VPN / network extension prompt.
3. Import your config and connect.

### Windows
1. Download the latest **EXE** from the [Releases page](https://github.com/azzphucdev/PlatformNext-Releases/releases).
2. Run the installer / binary (administrator rights may be required for TUN mode).
3. Import your config and connect.

### Linux
1. Download the latest Linux binary from the [Releases page](https://github.com/azzphucdev/PlatformNext-Releases/releases).
2. Make it executable and launch it (elevated privileges may be required for TUN mode).
3. Import your config and connect.
